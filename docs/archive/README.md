# Documentation Archive

**Last Updated:** 2025-11-02

This directory contains archived documentation from the Moldova Direct project. Archived documents are preserved for historical reference but are no longer actively maintained.

## Overview

Documentation is archived when:
- It's superseded by a newer version (e.g., annual reports)
- A feature is fully implemented or deprecated
- Information is time-bound and no longer current

See [ARCHIVAL_POLICY.md](../ARCHIVAL_POLICY.md) for complete archival guidelines.

## Archive Structure

```
archive/
└── 2025/
    ├── code-reviews/          # Annual code review reports
    ├── analysis-reports/      # Quality and coverage analyses
    └── update-summaries/      # Documentation update summaries
```

1. Check [DOCUMENTATION_INDEX.md](../../DOCUMENTATION_INDEX.md)
2. Search the repository: `grep -r "keyword" docs/`
3. Check git history: `git log --all --full-history -- "**/filename.md"`
