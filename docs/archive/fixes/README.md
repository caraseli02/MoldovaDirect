## Usage Notes

**File Naming Convention:**
- `{ISSUE}-{DESCRIPTION}.md` - Incident-specific fixes
- `{COMPONENT}-investigation-report.md` - Root cause analyses

**When to Archive:**
- Fixes older than 6 months should be moved to `/docs/archive/YYYY/`
- Keep recent fixes (< 6 months) for reference
