## Mobile Layout Changes

### Desktop (> 1024px)
- Hero: Full height with video
- Product carousel: 3-4 cards visible
- UGC gallery: 6 columns
- Collections: 2x2 grid
- Footer: Multi-column layout

### Tablet (768px - 1024px)
- Hero: Reduced height, video still visible
- Product carousel: 2-3 cards visible
- UGC gallery: 4 columns
- Collections: 2x2 grid
- Footer: 2 columns

### Mobile (< 768px)
- Hero: Single column, static image (no video)
- Product carousel: 1 card visible, swipe
- UGC gallery: 2 columns
- Collections: Stacked vertically
- Footer: Accordion sections
- Trust badges: 2 per row
- Stats: 2x2 grid
