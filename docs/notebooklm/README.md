## How to Use with NotebookLM

1. Go to [NotebookLM](https://notebooklm.google.com/)
2. Create a new notebook
3. Click "Add sources"
4. Upload all 9 markdown files from this folder
5. Start asking questions about the project

## Why This Bundle?

The full documentation has:
- 296 markdown files
- 100MB total (mostly images)
- Lots of archived/historical content
- Fragmented information across many files

This bundle:
- 9 focused files
- ~45KB total (text only)
- Current, accurate information
- Self-contained, no broken links
