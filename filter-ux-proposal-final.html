Pattern Overview

The Hybrid Sticky Filter combines the best of multiple patterns: always-accessible quick filters with an advanced bottom sheet for detailed filtering.

This pattern is used by industry leaders like Nike, Adidas, and H&M because it balances quick access with comprehensive filtering options while maintaining excellent mobile usability.

Key Features

Sticky Filter Bar

Quick-access filters remain visible while scrolling, eliminating the need to scroll back up to change filters.

Progressive Disclosure

Common filters in the sticky bar, advanced options in the bottom sheet. Users only see complexity when they need it.

Thumb-Friendly Design

All interactive elements positioned for easy thumb reach with minimum 44px touch targets.

Applied Filter Visibility

Active filters shown as removable chips with clear visual feedback and easy removal.

Quick Filter Actions

One-tap shortcuts for common filtering needs like "On Sale" or "New Arrivals".
